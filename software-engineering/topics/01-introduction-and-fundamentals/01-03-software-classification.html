1.3 What is Software / Classification

← Topic 1

A program is a step-by-step set of instructions written in a programming language to perform a specific task on a computer. Software is broader than a program because it includes the program itself along with proper documentation (such as requirements, design, coding, and testing records) and user or operating manuals that explain how the product is installed and used. In short:

Software = Program + Documentation (+ licensing for commercial products)

This distinction is important in exams. A student may write a working program for an assignment, but a commercial software product must also be documented, tested, and supported so that other developers and end users can understand and use it correctly over time.

Program vs Software Product

A program solves a narrow coding problem, while a software product is a complete deliverable intended for real-world use. The table below summarises the main differences:

ParameterProgramSoftware Product
DefinitionInstructions to achieve a specific taskProgram + documentation + licensing for commercial use
ScopeLimited to a specific problemBroader — solves real-world problems
StagesOne stage in software developmentFollows full lifecycle: feasibility, requirements, design, coding, testing
DocumentationMinimal or noneProper, complete documentation required

Classification of Software (by application)

Software can be grouped by the type of problem it solves. The main categories are:

  • System software: This category includes operating systems, compilers, editors, and device drivers that manage hardware and provide a platform on which other programs can run.
  • Application software: This software meets end-user needs and can be generic (such as MS Word for word processing) or customised for a specific client or organisation.
  • Networking and web applications: This software supports communication over networks and includes servers, security tools, and web technologies such as HTML, PHP, and XML.
  • Embedded software: This software is built into hardware devices (often stored in ROM) and controls products such as washing machines, microwaves, and satellites.
  • Business software: This software supports organisational operations such as inventory control, accounting, banking, hospital management, and stock market trading.
  • Entertainment and educational software: This category includes games, language translation tools, mapping applications, and other software used for learning or leisure.
  • Artificial intelligence software: This software uses techniques such as expert systems, neural networks, and pattern recognition to simulate intelligent behaviour.
  • Scientific and engineering software: This software supports specialised technical work, with examples such as MATLAB, AutoCAD, and SPICE for domain-specific calculations and design.
  • Utility software: This software provides supporting services such as antivirus protection, file compression, and voice recognition on a computer.
  • Document management software: This software helps organisations store documents digitally, control versions, and index content for easy retrieval.

Classification by copyright / license

Software can also be classified by how it is distributed and who owns the rights to use or modify it:

  • Commercial software: This software is sold under a license agreement, and the copyright remains with the company that developed it.
  • Shareware: This software is offered as a free trial, but the user must pay to continue using it after the trial period, and modification is usually not allowed.
  • Freeware: This software can be copied and distributed free of charge for non-commercial use, and modifications are often permitted by the author.
  • Public domain software: This software has no copyright protection, so anyone is free to copy, modify, and even reverse-engineer it without restriction.
Exam question — Program vs Software

Q: Differentiate between a program and software. Classify software with examples.

Model answer points:

  • A program is a set of instructions only, while software is a program together with documentation and, for commercial products, proper licensing.
  • Software can be classified by application into types such as system, application, embedded, business, and scientific software, and the answer should include one example for each type mentioned.
  • Software can also be classified by license into commercial, shareware, freeware, and public domain categories based on distribution rights and cost.
  • A program alone is not a complete software product because it lacks the documentation and manuals needed for maintenance and everyday use.
